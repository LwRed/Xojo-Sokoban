# Xojo-Sokoban
Portage de 5 niveaux du Jeu Sokoban sur Xojo (Xojo 2017 requis)

Fonctionnalités :
- Mac et Pc
- 8 Niveaux
- Plein écran 
- Chargement automatique des niveaux au format .txt
- Son
- Musique d'ambiance
- Animation du Personnage
