# Xojo-Sokoban
Portage de plusieurs niveaux du Jeu Sokoban sur Xojo (Xojo 2017 requis)

Fonctionnalités :
- Mac et Pc
- 16 Niveaux du jeu original (possibilité d'en créer d'autres)
- Plein écran 
- Chargement automatique des niveaux au format .txt
- Son
- Musique d'ambiance
- Animation du Personnage
